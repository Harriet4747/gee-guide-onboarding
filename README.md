# Gee Guide — published onboarding product

`published.json` is the shareable onboarding product served by the Gee Guide
Brick. It is generated (sanitized) from the author's local guide store and pushed
here on each publish. Installs fetch it at the raw URL:

    https://raw.githubusercontent.com/<owner>/<repo>/main/published.json

Do not hand-edit `published.json` — edit the source guide and republish.
